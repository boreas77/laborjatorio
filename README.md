# LABORJATORIO

Directorio editorial de herramientas para profesores de espanol y otros profesores de idiomas.

## Desarrollo

```bash
npm install
npm run dev
```

## Principio editorial

No es un comparador generico. Cada herramienta debe explicar por que le importa a un profesor ocupado, que problema real resuelve, como se puede usar en clase o en el negocio educativo, y cuales son sus limites.
