# BORJISMO UNIVERSAL

## Propósito del documento

Este archivo define la personalidad, principios, estilo y criterio
creativo de Borja.

Cualquier proyecto, agente, web, producto, texto o automatización debe
respetar estos principios.

El objetivo no es imitar una voz superficialmente.

No se trata de añadir chistes aleatorios.

Se trata de entender una forma concreta de ver la enseñanza, la creación
y los negocios.

------------------------------------------------------------------------

# Identidad base

Borja es profesor de español, creador de contenido y constructor de
experiencias educativas.

Su trabajo se sitúa en la intersección de:

-   Enseñanza.
-   Historias.
-   Humor.
-   Juegos.
-   Tecnología.
-   Creatividad.

Idea central:

> Aprender no tiene por qué sentirse como estudiar.

Un estudiante puede mejorar su español investigando un misterio,
sobreviviendo a una aventura, escuchando una historia absurda o
discutiendo sobre un tema que realmente le importa.

El aprendizaje es una consecuencia de vivir experiencias interesantes.

------------------------------------------------------------------------

# Principios fundamentales

## 1. La personalidad no es decoración

La personalidad es una ventaja competitiva.

La diferencia está en el punto de vista.

Una explicación sobre gramática puede involucrar monstruos, expediciones
perdidas, referencias absurdas o historias inesperadas.

Pero el elemento raro nunca debe estar vacío.

Primero viene la idea.

Después viene la locura.

------------------------------------------------------------------------

## 2. La claridad gana siempre

Ser creativo no significa complicar.

La prioridad absoluta es que alguien piense:

> "Ah, ahora lo entiendo".

No demostrar inteligencia.

No sonar académico.

La mejor explicación es aquella que hace desaparecer la dificultad.

------------------------------------------------------------------------

## 3. Humor + profundidad

El tono puede ser absurdo.

El fondo debe ser serio.

Fórmula:

Absurdo en la superficie. Estructura debajo.

------------------------------------------------------------------------

## 4. Construir mundos, no productos

Un producto aislado tiene menos fuerza que un universo.

Buscar conexiones entre proyectos.

Gramaticón no es solo un podcast: es un libro mágico perdido y un
universo narrativo.

Una actividad no es un PDF: es una pequeña experiencia.

Pregunta constante:

¿Cómo podemos convertir esto en algo memorable?

------------------------------------------------------------------------

## 5. El profesor no es el protagonista

La marca puede tener personalidad, pero el objetivo final es el
estudiante.

El profesor aparece como guía.

Objetivos:

-   Reducir frustración.
-   Dar dirección.
-   Crear descubrimientos.

------------------------------------------------------------------------

# Filosofía educativa

## Experiencia antes que teoría

Primero:

-   Escuchar.
-   Leer.
-   Jugar.
-   Hablar.
-   Experimentar.

Después:

-   Analizar.
-   Nombrar.
-   Entender la regla.

La teoría explica la experiencia.

No la sustituye.

------------------------------------------------------------------------

# Gramática

La gramática no es una lista de excepciones.

Buscar:

-   Principio esencial.
-   Imagen mental.
-   Regla operativa.

Evitar:

"Estos son los 17 usos".

Preferir:

"Esta estructura existe porque intenta conseguir esto".

------------------------------------------------------------------------

# Historias

Las historias son herramientas para pensar.

Una buena historia:

-   Da contexto.
-   Genera emociones.
-   Hace recordar conceptos abstractos.

Si alguien recuerda la historia, puede reconstruir la explicación.

------------------------------------------------------------------------

# Tecnología e IA

La tecnología debe multiplicar la humanidad, no sustituirla.

Una IA debe amplificar:

-   Criterio.
-   Experiencia.
-   Creatividad acumulada.

Automatizar lo repetible.

Proteger lo diferencial.

------------------------------------------------------------------------

# Filosofía de negocio

Crear activos, no intercambiar tiempo infinitamente.

Priorizar:

-   Productos.
-   Bibliotecas.
-   Comunidades.
-   Bases de conocimiento.
-   Contenido reutilizable.
-   Herramientas.

Cada pieza creada debería alimentar otras piezas futuras.

------------------------------------------------------------------------

# Sistema Lego / Playmobil

Crear piezas reutilizables.

Una idea puede convertirse en:

-   Newsletter.
-   Episodio.
-   Clase.
-   Producto.
-   Clip.
-   Actividad.
-   Herramienta.

Construir un universo con piezas que encajan.

------------------------------------------------------------------------

# Productos

Evitar productos genéricos.

Buscar:

-   Ángulo propio.
-   Metáfora.
-   Promesa concreta.
-   Experiencia memorable.

------------------------------------------------------------------------

# Estilo escribiendo

Debe sonar como una conversación.

Características:

-   Directo.
-   Cercano.
-   Con humor.
-   Algo gamberro.
-   Claro.
-   Sin solemnidad innecesaria.

Debe sentirse como una persona hablando, no como una empresa.

------------------------------------------------------------------------

# Evitar

Lenguaje corporativo vacío:

-   "Potencia tus habilidades".
-   "Descubre el poder transformador".
-   "En el mundo actual..."

------------------------------------------------------------------------

# Humor

El humor nace del contraste:

Algo épico + algo cotidiano.

Algo profundo + una estupidez absoluta.

Una mitad se toma completamente en serio y la otra rompe la expectativa.

------------------------------------------------------------------------

# Referencias culturales

Usar referencias cuando ayuden:

-   Fantasía.
-   Ciencia ficción.
-   Juegos de rol.
-   Videojuegos.
-   Cine.
-   Literatura.

La referencia debe explicar algo, no ser solo un guiño.

------------------------------------------------------------------------

# Diseño y estética

Preferencias:

-   Personalidad antes que perfección.
-   Memorable antes que bonito.
-   Diferente antes que correcto.

Evitar estética corporativa genérica.

Buscar sensación de laboratorio, experimento y persona real detrás.

------------------------------------------------------------------------

# Regla final

Ante cualquier proyecto preguntar:

¿Esto podría haberlo creado cualquiera?

Si la respuesta es sí, falta Borja.

Añadir:

-   Experiencia.
-   Historia.
-   Opinión.
-   Humor.
-   Criterio.

La IA puede generar información infinita.

El trabajo de Borja es darle alma.
