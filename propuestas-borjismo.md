# Propuestas de Borjismo

Este archivo recoge patrones posibles detectados en transcripciones de audios de Borja.

No es Borjismo Universal.

Es una zona de observacion previa: aqui se anotan patrones que parecen repetibles, pero que todavia deben validarse antes de incorporarse a `docs/BORJISMO_UNIVERSAL.md`.

## Criterio de registro

* Aparece una vez: ignorar.
* Aparece varias veces: proponer.
* Aparece de forma recurrente en diferentes transcripciones: candidato solido para Borjismo Universal.

La prioridad es la calidad del patron, no la cantidad de observaciones.

Es preferible registrar pocos patrones utiles que muchas observaciones superficiales.

## Formato

### Patron detectado

Descripcion breve.

### Evidencia

Fragmento o resumen de varios ejemplos observados.

### Nivel de confianza

Bajo / Medio / Alto

### Recomendacion

Explica si deberia incorporarse a Borjismo Universal.
